How PP is Calculated

Osu! PP is calculated using four core values:

  • Aim PP: Cursor movement accuracy (influenced by AR and mods)
  • Speed PP: Hit object density (influenced by DT/HT)
  • Accuracy PP: Hit timing precision (heavily weighted)
  • Strain PP: High intensity sections (streams, jumps)

Weightage Formula: Weighted PP = Raw PP × 0.95^(n-1)

Bonus PP: Up to 413.894 PP for setting scores on 1000 ranked maps

1. What is a PP Calculator?

A PP calculator is a tool that estimates how many performance points you will earn from a specific osu! beatmap. The calculator considers multiple factors, including your accuracy, combo, miss count, and selected mods. PP calculators are available as browser extensions, desktop applications, and command-line tools.

The most accurate PP calculators use the rosu-pp library, which is written in Rust and provides reliable performance calculations.

Some calculators also support all game modes, including osu!, taiko, mania, and catch. This makes them useful for players who play different modes and want consistent PP estimates across all of them. For detailed technical information about how PP is calculated, you can refer to the official osu! wiki on Performance Points.

2. How Does a PP Calculator Work?

Understanding how a PP calculator works helps you use it more effectively.

The Core Components

A PP calculator evaluates four key elements of your play:

  • Aim PP: This measures how accurately you move your cursor to hit notes. High approach rate (AR) and certain mods like Hidden (HD) or Flashlight (FL) affect aim difficulty.
  • Speed PP: This evaluates how quickly you can tap or click notes. Double Time (DT) significantly increases speed requirements.
  • Accuracy PP: This is based on how precisely you time your hits. Higher accuracy leads to significantly higher PP.
  • Strain PP: This accounts for high-intensity sections like streams, burst jumps, and complex patterns.

The Role of rosu-pp

Most modern PP calculators rely on the rosu-pp library, which provides accurate PP calculations for all modes. This library also powers many browser extensions and desktop tools.

3. Why Use a PP Calculator?

There are several practical reasons to use a PP calculator.

1. Estimate PP Before Playing

If you want to know whether a beatmap is worth farming, a PP calculator gives you a rough estimate before you even start playing. This saves time and helps you focus on maps that match your skill level.

2. Compare Different Mods and Accuracy Levels

You can test various mod combinations and see how they affect PP. For example, you can compare HD vs. HR vs. DT and decide which gives you the best potential score.

3. Plan Your Farming Strategy

Many players use PP calculators to plan their farming sessions. By estimating PP for multiple maps, you can create a list of beatmaps that fit your current rank and skill level.

4. Understand PP Weightage

The osu! PP system uses weightage, meaning your top scores are worth the most. A PP calculator helps you understand how your scores are weighted and where you can improve.

4. Types of PP Calculators

There are several ways to use a PP calculator. Each has its strengths.

1. Web-Based Calculators

Some calculators work directly in your browser. You enter the beatmap ID or upload the beatmap file, and the tool calculates PP. These are convenient because they require no installation.

2. Browser Extensions

Browser extensions are widely used. The most popular one has over 50,000 users and supports the latest 2025-03 rework. These extensions calculate PP directly on beatmap pages, making them very convenient.

3. Desktop Applications

Desktop apps offer more features, such as live in-play PP tracking and advanced mod options. Some apps also include a frameless always-on-top panel that shows PP during gameplay.

4. Command-Line Tools

Power users often prefer CLI tools that can batch-process beatmaps and provide detailed output. These are useful for automating PP calculations.

5. Developer Libraries

For advanced users, libraries like rosu-pp-py allow you to build custom PP calculators with Python. This provides full control and deep customization.

5. Step-by-Step Guide to Using a PP Calculator

If you are new to PP calculators, follow these steps to get started.

  1. Choose Your Calculator: Select a tool that fits your needs. For most players, a browser extension or web-based tool is best.
  2. Enter Beatmap Details: Provide the beatmap ID or upload the beatmap file.
  3. Select Game Mode: Choose the correct mode for your play.
  4. Select Mods: Toggle mods like HD, HR, DT, or FL.
  5. Enter Performance Inputs: Add your accuracy, combo, and miss count.
  6. Add Ranked Scores: Enter your number of ranked scores to calculate bonus PP.
  7. Click Calculate: The tool will show your estimated PP along with a breakdown.

6. Common Mistakes When Using a PP Calculator

Even experienced players make mistakes with PP calculators. Here are the most common errors to avoid.

Mistake 1: Using the Wrong Mode

Always ensure you select the correct game mode. PP values vary significantly between osu!, taiko, mania, and catch.

Mistake 2: Incorrect Beatmap File

If you are using a file-based calculator, make sure you are using the correct .osu file. An outdated or modified file can give inaccurate results.

Mistake 3: Forgetting Misses and Combo

Some players forget to enter misses and combo. Both factors significantly affect PP. Always include them for accurate estimates.

Mistake 4: Comparing Different PP Reworks

PP calculations change with each rework. Avoid comparing PP values from different versions, as they are not directly comparable.
